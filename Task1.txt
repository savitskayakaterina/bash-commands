* Определить имя папки, в которой вы находитесь - pwd
* Создать папку с именем test1 - mkdir test1
* Перейти в папку test1 - cd test1
* Создать файл 1,2 и 3 внутри каталога test1 - touch 1.txt 2.txt 3.txt
* Проверить содержимое каталога test1 - ls test1
* Создать папку test2 внутри домашней директории - mkdir ~/test2
* Удалить папку test2 - rm ~/test2
* Удалить файл 2 - rm 2.txt
* Создать папку test3 и добавить в нее два файла
mkdir test3
touch test3/file1 test3/file2
* Удалить папку test3 - rm -r test3
* Создать папку test4 - mkdir test4
* Переместить файлы 1 и 3 в папку test4 - mv 1.txt 3.txt test4/
* Добавить в файл 1 три строки со словами line
echo "line1" >> 1.txt
echo "line2" >> 1.txt
echo "line3" >> 1.txt
* Посмотреть содержимое файла 1 - cat 1.txt
* Добавьте в файл 3 три строки со словами line
echo "line1" >> 3.txt
echo "line2" >> 3.txt
echo "line3" >> 3.txt
* Просмотрите содержимое двух файлов (1 и 3) сразу
cat 1.txt 3.txt
* Используя один из редакторов замените все строки
vim 1.txt
